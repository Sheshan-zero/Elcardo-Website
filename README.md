# Elcardo Industries - Premium React Website

A modern, premium React-based website for Elcardo Industries, designed with inspiration from global leaders like Apple, Tesla, Siemens, and Lamborghini.

## Features

- **3D Interactive Hero**: Built with React Three Fiber, featuring a rotating torus knot, orbiting rings, and floating particles.
- **Premium Animations**: Smooth scrolling with Lenis, scroll-triggered reveals with Framer Motion, and Lamborghini-style image preview panels.
- **Glassmorphism Design**: Frosted glass effects on the navigation bar and innovation cards.
- **Dynamic Content**: Auto-cycling hero slideshow, interactive company portfolio, and scrolling client tickers.

## Tech Stack

- **React + Vite**: Fast, modern development environment.
- **Framer Motion**: For elegant, physics-based UI transitions.
- **React Three Fiber & Drei**: For high-performance WebGL 3D graphics.
- **Lenis Scroll**: For butter-smooth, Apple-like scrolling physics.
- **Vanilla CSS Variables**: For a scalable, easily-customizable design system.

## Setup & Local Development

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   *The site will be available at `http://localhost:5173`*

3. **Build for Production**
   ```bash
   npm run build
   ```
